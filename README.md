# ZZZ-registration-helper

就是一个ZZZ抢码小脚本

## 如何使用

- ZZZ客户端，设置为中文
- 直接运行Release编译好的代码或自行编译，**一定要使用管理员运行!!!**
- 已适配全部屏幕，但我个人推荐您以**窗口模式**运行ZZZ

## 开始使用

首先打开终端，安装相应文件

```python
pip install -r requirements.txt
```

之后以**管理员身份**打开终端，运行文件

```python
python main.py
```

根据命令行的提示操作，出错了可以直接**Ctrl+C**大法

如果您发现任何bug，请提交issue
